Para agregar:

1)git add X ---- siendo X el nombre del archivo 
2)git commit -m 'Creacion de X'---- Para que se sepa el porque del cambio
	2.1)git commit --amend -m "nuevo mensaje"----Si es que te equivocas en el mensaje antes de subirlo
3)git push---- para que se agregue

Si es que push no funca no desesperes. El primer push que hacemos nos pedirá
que definamos cual es el remoto al que queremos subir el contenido, por eso debemos
 indicarle origin, y además la branch que será master. No sabés que es una branch aún, pero 
cuando termines este curso te prometo que entenderás todo lo que dice este comando:

0)git push -u origin master
